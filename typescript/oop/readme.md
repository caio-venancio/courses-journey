- [] Para que serve interface em typescript?
    As interfaces no TypeScript servem para definir contratos de estrutura (tipos) para objetos, garantindo que eles sigam um formato específico com propriedades e métodos definidos. Elas aumentam a segurança do código durante o desenvolvimento, oferecem autocompletar eficiente e melhoram a legibilidade, desaparecendo após a compilação para JavaScript.

- [] O que acontece se der console.log de um objeto em javascript?
    Ao utilizar console.log() em um objeto instanciado de uma classe TypeScript (que é compilado para JavaScript), o console exibirá uma representação estruturada do objeto naquele momento, permitindo inspecionar suas propriedades e métodos.
