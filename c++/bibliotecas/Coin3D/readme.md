vcpkg é útil?
Sim, o vcpkg é extremamente útil, especialmente para desenvolvedores C/C++, pois é um gerenciador de pacotes que simplifica a instalação, gerenciamento e controle de versões de bibliotecas open-source, automatizando dependências e configurando projetos com CMake, economizando tempo e evitando conflitos de compilação no Windows, macOS e Linux.
