| Algoritmo                  | Complexidade | Estruturas Aux.             | Ideia do método                                                   |
| -------------------------- | ------------ | --------------------------- | ----------------------------------------------------------------- |
| Mergesort                  | O(n log n)   | Vetor auxiliar              | Divide o vetor em duas partes, ordena recursivamente e intercala  |
| Contagem de inversões      | O(n log n)   | Vetor auxiliar              | Usa a intercalação do mergesort para contar inversões             |
| Seleção em tempo linear    | O(n)         | Particionamento recursivo   | Usa mediana das medianas para escolher um pivô balanceado         |
| Karatsuba                  | O(n^1,58)    | Chamadas recursivas         | Reduz multiplicações usando identidades algébricas                |
| Strassen                   | O(n^2,81)    | Matrizes auxiliares         | Multiplica matrizes reduzindo o número de multiplicações          |
| Par de pontos mais próximo | O(n log n)   | Vetores ordenados por x e y | Divide os pontos, resolve recursivamente e verifica faixa central |
