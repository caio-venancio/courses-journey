| Algoritmo                 | Complexidade     | Estruturas Aux.  | Ideia do método                            |
| --------------------------| ---------------- | ---------------- | ------------------------------------------ |
| Interval Scheduling       | O(n log n)       | array            | Escolher o que termina primeiro            |
| Interval Partitioning     | O(n log n)       | minheap          | Profundidade de intervalos abertos em qualquer tempo |
| Minimize Max lateness     | O(n log n)       | array            | Deadine mais próxima primeiro | 
| Knapsack fracionária      | O(n log n)       | array            | Ordem decrescente de razão valor/peso |
| Troco de moeda            | O(n)             | array            | Pegar o máximo da maior moeda e depois passar para próxima |
| Algoritmo do caminhoneiro | O(n log n)       | array            | Dirigir o máximo até o posto onde não poderia continuar |
| Código de Huffman         | O(n log n)       | minheap          | Códigos menores para caractere frequentes |
